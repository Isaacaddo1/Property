BlueJ Project 'Project Viewer'

This project displays the properties available on an airbnb using a GUI.

Purpose of the project
=======================

The purpose of this project was to develop understanding of method calling from
other classes, use of if and else statements, updating fields and returning values.

How to use this project
=======================

Open the PropertyViewer Class where the first property's details in the portfolio are displayed.
Clicking next and previous will display the next property and previous property on the portfolio 
respectively.
Clicking the Toggle Favourite button will save that property as your favourite property.
Clicking View Property on Map will open the location of the property currently being 
viewed on google maps.
